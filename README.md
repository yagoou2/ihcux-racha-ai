## 📌 Visão Geral
O **RachaAí** é um aplicativo móvel projetado para simplificar a gestão de despesas em grupo. Ele permite que os usuários criem grupos, adicionem despesas compartilhadas, dividam custos e acompanhem pagamentos — eliminando confusões e evitando aquelas situações embaraçosas de "quem deve o quê".

> 💡 “Somos anti-muquiranice.”  

---

## 🚀 Funcionalidades

- 👥 Criar e gerenciar grupos (ex: viagens, festas, casas compartilhadas)
- 💰 Adicionar e dividir despesas entre os participantes
- 📊 Visualizar o detalhamento de custos
- 📷 Escanear recibos usando a câmera
- 💳 Acompanhar dívidas e pagamentos
- ✅ Confirmar transações
- 📩 Feedback/mensagem opcional após o pagamento

---

## 🧭 Telas do App (Fluxo)

### 🏠 Tela 1 – Dashboard
- Lista de grupos ativos  
- Visão geral do saldo consolidado  

### 📄 Tela 2 – Detalhes do Grupo
- Ver despesas dentro de um grupo  
- Ver participantes e seus saldos  

### ➕ Tela 3 – Adicionar Despesa
- Entrada:
  - Valor
  - Descrição
  - Participantes
- Escolher como dividir a despesa  

### 📷 Tela 4 – Scanner
- Escanear recibos usando a câmera  
- Opção para fazer upload ou inserir código  

### 💸 Tela 5 – Resumo de Dívidas
- Mostra:
  - Quem deve para quem  
  - Valor total a pagar  
- Botão para prosseguir com o pagamento  

### ✅ Tela 6 – Confirmação de Pagamento
- Exibe:
  - Sucesso no pagamento  
  - Detalhes da transação  
- Campo de mensagem opcional

## 📊 Exemplo de Uso

1. Criar um grupo: *“Viagem para a praia”* 🏖️  
2. Adicionar despesas (comida, gasolina, hospedagem)  
3. Dividir custos entre amigos  
4. O app calcula as dívidas automaticamente  
5. Os usuários pagam diretamente e confirmam  
6. Todo mundo fica quite 🤝  

---

## 🎯 Objetivo

O RachaAí visa:
- Reduzir o atrito em despesas compartilhadas  
- Prevenir mal-entendidos  
- Tornar a colaboração financeira simples e transparente
